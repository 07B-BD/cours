# Module 1 — Introduction

## Objectifs du module
- Expliquer à quoi sert une base de données dans un système logiciel.
- Distinguer formats de données, base de données, SGBD et moteur de stockage.
- Décrire la structure d’une base relationnelle (tables, champs, enregistrements, clés, index, relations).
- Lire un mini-modèle relationnel et anticiper les tables/relations principales.
